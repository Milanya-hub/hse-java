# Commander: Двухпанельный Файловый Менеджер
tag - commander

## Цель
Реализовать файловый менеджер с двумя панелями (левая и правая).

Функции:
1. Показ содержимого текущей директории в каждой панели.
2. Первый элемент списка всегда `...` для перехода на уровень вверх.
3. Copy - копирование выбранного файла в противоположную панель.
4. Move - перемещение выбранного файла в противоположную панель.
5. Remove - удаление выбранного файла.

## UI Требования (JavaFX)

1. Два списка: `Left` и `Right`.
2. В обоих списках первым элементом отображается `...`.
3. При клике на `...` панель переходит к родительской директории текущего пути.
4. Кнопки:
`Copy` копирует файл из активной панели в противоположную.
`Move` переносит файл из активной панели в противоположную.
`Delete` удаляет файл из активной панели.

## Тестирование

Тесты проверяют само приложение: поднимается JavaFX-окно, эмулируются действия пользователя (клики по спискам и кнопкам). Ожидаемое поведение проверяется по состоянию файловой системы и отображению интерфейса.

## Сценарии автотестирования

Базовые (`@Tag("commander")`):
1. Список панели содержит `...` как первый элемент.
2. Переход в папку и возврат через `...`.
3. `copy` создаёт копию в противоположной панели и не удаляет исходник.
4. `move` переносит файл (в исходной панели файл исчезает).
5. `delete` удаляет файл.
